Merging Indonesia's PODES Waves

PODES 1993–2014 Merge

This folder contains crosswalks and do-files that merge villages across the Indonesian village census (PODES) waves of 1993, 1996, 2000, 2003, 2005, 2008, 2011, and 2014. The do-files also merge PODES with the village population counts from the 2010 population census. The folder also includes crosswalks spanning 2013–2021 for users who wish to merge PODES 2014 with PODES 2018–2021.

If you use this code in your own work, please cite the following paper:

Cassidy, Traviss (2025): "Revenue Persistence and Public Service Delivery," Economic Journal, 135 (670): 2017–2053, 2025.

For details on the merge procedure, see the Data Appendix in the paper cited above.
